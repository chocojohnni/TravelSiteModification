Events Web API

This API provides event discovery and booking features for partner travel sites.

Base URL: https://cis-iis2.temple.edu/Fall2025/3342_tup84792/WebAPI/

Endpoints

  • GET /api/event/agencies?city=&state=
  • GET /api/event/activities?city=&state=
  • GET /api/event/find?city=&state=&fromDate=&toDate=&minPrice=&maxPrice=&keyword=
  • GET /api/event/findbyvenue?venueId=&city=&state=&fromDate=&toDate=
  • POST /api/event/reserve
  • POST /api/token/create

Return Fields

Typical DataSet columns returned:

  • AgencyId, AgencyName, Phone, Email
  • EventId, EventName, EventDateTime, Description
  • VenueId, VenueName, Price, SeatsAvailable
